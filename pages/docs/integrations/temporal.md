# Temporal Integration

This guide explains how to use the Temporal integration (`forze_temporal`) with Forze.

## Prerequisites

- Temporal.io server
- `forze[temporal]` extra installed (includes `temporalio`)

## Overview

The `forze_temporal` package provides integration with Temporal.io for orchestrating workflows and activities within your Forze backend services.

### Usage

Currently, the `forze_temporal` package provides the foundational bindings and compatibility layers for `temporalio`. More features, adapters, and patterns will be detailed as the integration stabilizes.

*Note: The Temporal integration is currently experimental.*
