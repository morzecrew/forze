# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please do not open a public issue.

Instead, report it privately to: misery7100@gmail.com

Please include:

- a description of the issue
- steps to reproduce it
- possible impact
- any suggested mitigation, if available

I will try to acknowledge the report within a reasonable time and work on a fix as appropriate.

## Supported Versions

Security fixes are generally provided for the latest released version.
